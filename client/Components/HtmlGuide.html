Introduction to HTML

What is HTML?
HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML uses elements represented by tags to define text, links, images, and more.
What are Tags?
Tags are keywords wrapped in angle brackets (e.g., <p>). Most HTML elements have an opening tag, content, and a closing tag (e.g., <p>This is a paragraph</p>).
Doctype Declaration
The <!DOCTYPE html> declaration defines the document as HTML5. It should be placed at the very beginning of every HTML document.
What is an HTML element?
An HTML element is a part of an HTML document, defined by a start tag, content, and an end tag.

HTML Document Structure

The <html> Element
The <html> element is the root of every HTML document. It wraps all content, including the head and body sections.
The <head> Element
The <head> element contains meta-information about the document, such as the title, character encoding, stylesheets, and scripts.
The <body> Element
The <body> element holds all the visible content of the webpage—headings, paragraphs, images, links, and more.
Block vs Inline Elements
Block elements start on a new line and stretch the full width. Inline elements only take up as much width as necessary.
Common Elements
Examples: <div>, <p>, <span>, <a>, <h1>–<h6>, <ul>, <li>, etc.

Semantics

What are semantic elements?
Semantic elements clearly describe their meaning in a human- and machine-readable way (e.g., <header>, <footer>, <main>, <article>).
Benefits of Semantic HTML
Improves accessibility, SEO, and code readability by providing meaningful structure.
Common Semantic Tags
<header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.

Text Elements

Headings (<h1>–<h6>)
HTML provides six levels of headings: <h1> (most important) to <h6> (least). They help structure the content hierarchically.
Paragraphs (<p>)
The <p> element represents a paragraph of text. Browsers add space above and below paragraphs automatically.
Bold and Emphasis
Use <strong> for strong importance (usually bold) and <em> for emphasized text (usually italic). Avoid using <b> or <i> for meaning.

Lists

Unordered Lists (<ul>)
An unordered list (<ul>) displays bullet points. Each item is placed in a <li> (list item) element.
Ordered Lists (<ol>)
Ordered lists (<ol>) display items in a numbered sequence. Like unordered lists, each item uses <li>.
Definition Lists (<dl>)
A definition list uses <dl> with <dt> (definition term) and <dd> (definition description). Useful for glossaries or terms.

Links

The <a> Tag
The <a> (anchor) element defines a hyperlink. Use the href attribute to specify the destination URL.
Example: <a href="https://example.com">Visit</a>
Opening Links in a New Tab
Add target="_blank" to an anchor to open the link in a new browser tab.
Example: <a href="..." target="_blank">New Tab</a>
Linking to Page Sections
Use #id in the href to jump to an element with that ID on the same page.
Example: <a href="#footer">Go to Footer</a>

Images and Media

Inserting Images
Use the <img> tag with src and alt attributes to embed an image.
Example: <img src="image.jpg" alt="Description">
Alt Text
The alt attribute provides alternative text if the image can't load. It also helps with accessibility and SEO.
Embedding Video
Use the <video> element with a src or <source> inside it.
Example: <video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support video.
</video>
Embedding Audio
Use the <audio> tag with controls to include sound clips.
Example: <audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>

Tables

Creating a Table
Tables are created using the <table> element with nested rows and cells: <table>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
Table Headers
Use <th> inside <tr> to create header cells. These are usually bold and centered.
Table Structure Elements
HTML tables can include:
  • <thead> – Table header group
  • <tbody> – Table body rows
  • <tfoot> – Table footer group
These help organize and style tables more cleanly.

Forms and Inputs

The <form> Element
The <form> tag defines an HTML form used to collect user input. It commonly uses method (e.g., "get" or "post") and action (where to send data).
Input Types
<input> can accept various types:
  • text
  • email
  • password
  • checkbox
  • radio
  • submit
  • file, number, date, etc.
Form Controls
Common form elements include:
  • <input> — for basic fields
  • <textarea> — for multi-line input
  • <select> with <option> — for dropdowns
  • <button> — for actions like submit/reset
Labels and Accessibility
Use <label> with for="inputID" to make forms accessible and improve usability.
Example: <label for="email">Email:</label><input id="email">

Semantic HTML

What is Semantic HTML?
Semantic HTML uses meaningful tags that describe the purpose of the content, not just how it looks. This improves accessibility, SEO, and code clarity.
Common Semantic Tags
These tags provide structure and meaning:
  • <header> — for introductory content
  • <nav> — for navigation links
  • <main> — for the main content
  • <section> — for grouped content
  • <article> — for independent, reusable content
  • <aside> — for tangential info or sidebars
  • <footer> — for page or section footers
Why Use Semantic Tags?
- They help screen readers interpret content properly.
- They make your HTML easier to read and maintain.
- They improve how search engines understand your page layout.

HTML Attributes

What are Attributes?
Attributes provide extra information about elements. They are added inside the opening tag and usually follow the name="value" pattern.
Example: <a href="https://example.com">
Global Attributes
These can be used on any HTML element:
  • id — uniquely identifies an element
  • class — applies CSS or JS hooks
  • style — inline CSS
  • title — tooltip text
  • hidden — hides the element
  • data-* — custom attributes for storing data
Boolean Attributes
Boolean attributes don’t require a value. Their presence alone implies "true".
Examples: checked, disabled, required, readonly

Meta Tags & The <head> Element

What is the <head> Element?
The <head> contains metadata about the document: not visible to users, but crucial for the browser and search engines.
Essential Meta Tags
- <meta charset="UTF-8"> — Sets the character encoding
- <meta name="viewport" content="width=device-width, initial-scale=1.0"> — Controls responsiveness on mobile
- <meta name="description" content="Page description"> — Helps with SEO
- <meta name="author" content="Your Name"> — Credits the author
Other Common Head Elements
- <title> — Sets the tab name
- <link rel="stylesheet"> — Links external stylesheets
- <script> — Links or contains JavaScript
- <base> — Sets a base URL for all relative URLs

Inline vs Block Elements

Block Elements
Block elements take up the full width of their container and always start on a new line.
Examples: <div>, <p>, <h1> to <h6>, <ul>, <table>
Inline Elements
Inline elements do not start on a new line and only take up as much width as necessary.
Examples: <span>, <a>, <strong>, <em>, <img>
Inline-Block Elements
These behave like inline elements (flow inline) but allow you to set width, height, margin, etc., like block elements. Often used in CSS.
Tip: Apply with display: inline-block in CSS.

Comments & Escaping Characters

HTML Comments
Use comments to explain or hide parts of your HTML code.
Syntax: <!-- This is a comment -->
Comments are not shown in the browser and are helpful for documentation or debugging.
Escaping Special Characters
Some characters must be "escaped" in HTML to be displayed properly:
  • &amp; → &
  • &lt; → <
  • &gt; → >
  • &quot; → "
  • &nbsp; → non-breaking space
Use these when writing code examples or displaying symbols.

iframes & Embedding Content

What is an <iframe>?
The <iframe> tag embeds another HTML page inside the current one.
Example: <iframe src="https://example.com" width="600" height="400"></iframe>
Security Considerations
Using <iframe> can raise security issues. Use the sandbox attribute to restrict behavior and enhance safety.
Example: <iframe src="..." sandbox="allow-scripts"></iframe>
Embedding YouTube Videos
You can use an iframe to embed YouTube content directly:
<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

HTML5 APIs (Overview)

What Are HTML5 APIs?
HTML5 introduced built-in browser APIs that work closely with JavaScript to add powerful features to web pages. These APIs extend HTML beyond markup.
Geolocation API
Allows web applications to access the user's physical location (with permission). Often used in maps and weather apps.
Accessed using: navigator.geolocation
Web Storage API
Lets you store data in the browser:
  • localStorage — persists even after closing the browser
  • sessionStorage — lasts until the tab is closed
Both use key-value pairs and are more secure & flexible than cookies.
Canvas API
The <canvas> element allows for dynamic, scriptable rendering of graphics (charts, games, etc.) using JavaScript.
Example use: drawing 2D shapes, graphs, or visualizations.
