Introduction to HTML

What is HTML?
HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML uses elements represented by tags to define text, links, images, and more.
What are Tags?
Tags are keywords wrapped in angle brackets (e.g., <p>). Most HTML elements have an opening tag, content, and a closing tag (e.g., <p>This is a paragraph</p>).
Doctype Declaration
The <!DOCTYPE html> declaration defines the document as HTML5. It should be placed at the very beginning of every HTML document.
What is an HTML element?
An HTML element is a part of an HTML document, defined by a start tag, content, and an end tag.

HTML Document Structure

The <html> Element
The <html> element is the root of every HTML document. It wraps all content, including the head and body sections.
The <head> Element
The <head> element contains meta-information about the document, such as the title, character encoding, stylesheets, and scripts.
The <body> Element
The <body> element holds all the visible content of the webpage—headings, paragraphs, images, links, and more.
Block vs Inline Elements
Block elements start on a new line and stretch the full width. Inline elements only take up as much width as necessary.
Common Elements
Examples: <div>, <p>, <span>, <a>, <h1>–<h6>, <ul>, <li>, etc.

Semantics

What are semantic elements?
Semantic elements clearly describe their meaning in a human- and machine-readable way (e.g., <header>, <footer>, <main>, <article>).
Benefits of Semantic HTML
Improves accessibility, SEO, and code readability by providing meaningful structure.
Common Semantic Tags
<header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.

Attributes

What are HTML attributes?
Attributes provide additional information about HTML elements and are always included in the opening tag.
Global Attributes
Global attributes can be used on any HTML element (e.g., id, class, style, title).
Boolean Attributes
Boolean attributes are those that can only have a true/false value, such as checked, disabled, required.

Text Elements

Headings (<h1>–<h6>)
HTML provides six levels of headings: <h1> (most important) to <h6> (least). They help structure the content hierarchically.
Paragraphs (<p>)
The <p> element represents a paragraph of text. Browsers add space above and below paragraphs automatically.
Bold and Emphasis
Use <strong> for strong importance (usually bold) and <em> for emphasized text (usually italic). Avoid using <b> or <i> for meaning.

Lists

Unordered Lists (<ul>)
An unordered list (<ul>) displays bullet points. Each item is placed in a <li> (list item) element.
Ordered Lists (<ol>)
Ordered lists (<ol>) display items in a numbered sequence. Like unordered lists, each item uses <li>.
Definition Lists (<dl>)
A definition list uses <dl> with <dt> (definition term) and <dd> (definition description). Useful for glossaries or terms.

Links

The <a> Tag
The <a> (anchor) element defines a hyperlink. Use the href attribute to specify the destination URL.
Example: <a href="https://example.com">Visit</a>
Opening Links in a New Tab
Add target="_blank" to an anchor to open the link in a new browser tab.
Example: <a href="..." target="_blank">New Tab</a>
Linking to Page Sections
Use #id in the href to jump to an element with that ID on the same page.
Example: <a href="#footer">Go to Footer</a>

Images and Media

Inserting Images
Use the <img> tag with src and alt attributes to embed an image.
Example: <img src="image.jpg" alt="Description">
Alt Text
The alt attribute provides alternative text if the image can't load. It also helps with accessibility and SEO.
Embedding Video
Use the <video> element with a src or <source> inside it.
Example: <video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support video.
</video>
Embedding Audio
Use the <audio> tag with controls to include sound clips.
Example: <audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>

Tables

Creating a Table
Tables are created using the <table> element with nested rows and cells: <table>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
Table Headers
Use <th> inside <tr> to create header cells. These are usually bold and centered.
Table Structure Elements
HTML tables can include:
  • <thead> – Table header group
  • <tbody> – Table body rows
  • <tfoot> – Table footer group
These help organize and style tables more cleanly.

Forms

The <form> Element
The <form> element is used to collect user input and submit it to a server.
Input Types
Input types include text, password, email, checkbox, radio, submit, and more.
Form Controls
Form controls include <input>, <textarea>, <select>, <button> and others.
