Introduction to CSS

What is CSS?
CSS stands for Cascading Style Sheets. It controls how HTML elements are displayed, including layout, colors, fonts, and spacing.
How CSS Works
CSS applies rules to HTML using selectors. The browser interprets these styles and applies them in a cascading order.

Selectors

Basic Selectors
  • * — universal selector
  • p — element selector
  • .class — class selector
  • #id — ID selector
Combinators
  • div p — descendant
  • div > p — direct child
  • div + p — adjacent sibling
  • div ~ p — general sibling

Box Model

What is the Box Model?
Every HTML element is a rectangular box composed of: content, padding, border, and margin.
