Introduction to CSS

CSS, or Cascading Style Sheets, is the language that describes the style of an HTML document. It controls the visual presentation, including layout, colors, fonts, spacing, and responsiveness across devices.

By separating content (HTML) from presentation (CSS), developers can maintain cleaner code and apply consistent styles across multiple pages.

Selectors

CSS selectors are patterns used to target and style specific HTML elements. They allow you to apply styles based on element names, classes, IDs, attributes, and relationships between elements.

  • Element Selector: Targets elements by tag name. Example: p targets all paragraphs.
  • Class Selector: Targets elements by class. Example: .button targets elements with class "button".
  • ID Selector: Targets a unique element by ID. Example: #header.
  • Attribute Selector: Targets elements with specific attributes. Example: input[type="text"].
  • Descendant Selector: Targets elements inside another. Example: div p.
  • Child Selector: Direct child elements. Example: ul > li.
  • Group Selector: Style multiple selectors together. Example: h1, h2, h3.

Colors & Units

CSS provides various ways to define colors and units for styling elements, allowing precise control over the appearance and layout.

Color Values:

  • Named Colors: Example: red, blue, green.
  • Hexadecimal: Example: #ff0000 for red.
  • RGB: Example: rgb(255, 0, 0).
  • RGBA: Adds transparency: rgba(255, 0, 0, 0.5).
  • HSL: Hue, Saturation, Lightness: hsl(0, 100%, 50%).

Units:

  • Pixels (px): Absolute size unit.
  • Percent (%): Relative to the parent element.
  • Em: Relative to the font-size of the parent.
  • Rem: Relative to the root elementâ€™s font-size.
  • Viewport Units: vw (viewport width), vh (viewport height).

Box Model

The CSS Box Model describes how elements are structured and spaced in a page layout. Each element is represented as a rectangular box composed of the following areas:

  • Content: The innermost part where text or other content appears.
  • Padding: Space between the content and the border. Adds inner spacing.
  • Border: The outline surrounding the padding (or content if no padding is set).
  • Margin: Space outside the border that separates the element from other elements.

Understanding the box model is crucial for proper element sizing and layout control. Total width of an element is calculated as: 

          Total Width = content width + padding + border + margin

Typography

Typography in CSS controls the style, appearance, and readability of text on a webpage. CSS offers various properties to customize fonts and text layout.

Common Typography Properties:

  • font-family: Defines the typeface. Example: font-family: Arial, sans-serif;
  • font-size: Sets the size of the text. Example: font-size: 16px;
  • font-weight: Controls the thickness of the text. Example: font-weight: bold;
  • line-height: Adjusts the spacing between lines of text.
  • letter-spacing: Controls spacing between characters.
  • text-align: Aligns text horizontally. Options: left, right, center, justify.
  • text-decoration: Adds styling like underline, overline, or line-through.
  • color: Sets the text color.

Choosing the right typography enhances user experience and accessibility on the web.

Positioning

CSS positioning allows you to control the placement of elements on the page. Different positioning schemes help create complex layouts and control stacking order.

Position Values:

  • static: Default position; elements flow naturally in the document.
  • relative: Positioned relative to its normal position; can be offset with top, right, bottom, left.
  • absolute: Positioned relative to the nearest positioned ancestor (non-static).
  • fixed: Positioned relative to the viewport; stays in place during scrolling.
  • sticky: Toggles between relative and fixed based on scroll position.

Example Usage:

          position: absolute;
          top: 20px;
          left: 50px;

Additionally, z-index controls the stacking order of overlapping elements.

Flexbox

Flexbox is a CSS layout model that makes it easy to design flexible and responsive layout structures without using floats or positioning.

Core Properties:

  • display: flex; Enables flex context for a container.
  • flex-direction: Defines the main axis direction (row, column).
  • justify-content: Aligns items along the main axis (e.g., center, space-between).
  • align-items: Aligns items along the cross axis (e.g., stretch, center).
  • flex-wrap: Allows items to wrap onto multiple lines.
  • align-self: Aligns a specific item differently from others.

Example:

        .container {
            display: flex;
            justify-content: center;
            align-items: center;
        }

Flexbox is ideal for aligning elements in both horizontal and vertical axes within a container.

Grid

CSS Grid Layout is a powerful system for creating two-dimensional layouts on the web. It allows you to design complex grids with precise placement of items.

Core Properties:

  • display: grid; Establishes a grid layout on a container.
  • grid-template-columns: Defines the number and size of columns.
  • grid-template-rows: Defines the number and size of rows.
  • gap: Sets spacing between rows and columns.
  • grid-column: Specifies the column span of an item.
  • grid-row: Specifies the row span of an item.
  • justify-items / align-items: Align items within grid cells.

Example:

        .container {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 10px;
        }

Grid layout is ideal for building page layouts, galleries, and dashboards where both rows and columns are needed.

Transitions & Animations

CSS Transitions and Animations allow you to add smooth changes and dynamic effects to your web elements, enhancing user experience.

Transitions:

Transitions enable smooth changes between property values over time. 

        .element {
            transition: background-color 0.3s ease-in-out;
        }

        .element:hover {
            background-color: #3498db;
        }

Animations:

Animations allow complex, multi-step movements defined using @keyframes. 

              @keyframes bounce {
                  0%, 100% { transform: translateY(0); }
                  50% { transform: translateY(-20px); }
              }

              .element {
                  animation: bounce 2s infinite;
              }

Use animations for loading indicators, attention cues, and more.

Responsive Design

Responsive Design ensures your website looks and works well on all devices, from desktops to mobile phones, by adapting layout and content.

Media Queries:

Media queries apply CSS rules based on device characteristics like screen width. 

          @media (max-width: 768px) {
              .container {
                  flex-direction: column;
              }
          }

Fluid Layouts & Units:

Use relative units like %, em, and vw to create flexible designs.

Mobile-First Approach:

Design for smaller screens first, then add enhancements for larger screens.

Variables (Custom Properties)

CSS Variables, also called Custom Properties, allow you to store reusable values and simplify theme management.

Declaring Variables:

          :root {
              --primary-color: #3498db;
              --font-size: 16px;
          }

Using Variables:

          button {
              background-color: var(--primary-color);
              font-size: var(--font-size);
          }

Variables can be overridden in specific scopes and support fallback values.
