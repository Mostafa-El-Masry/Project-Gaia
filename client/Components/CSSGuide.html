Introduction to CSS

What is CSS?
CSS stands for Cascading Style Sheets. It controls how HTML elements are displayed, including layout, colors, fonts, and spacing.
How CSS Works
CSS applies rules to HTML using selectors. The browser interprets these styles and applies them in a cascading order.

Selectors

Basic Selectors
  • * — universal selector
  • p — element selector
  • .class — class selector
  • #id — ID selector
Combinators
  • div p — descendant
  • div > p — direct child
  • div + p — adjacent sibling
  • div ~ p — general sibling

Box Model

What is the Box Model?
Every HTML element is a rectangular box composed of: content, padding, border, and margin.

Positioning

Position Property
The position property defines how an element is placed:
  • static (default)
  • relative — positioned relative to itself
  • absolute — positioned relative to the nearest positioned ancestor
  • fixed — positioned relative to the viewport
  • sticky — toggles between relative and fixed

Flexbox

What is Flexbox?
Flexbox is a layout model designed for arranging items in a row or column efficiently. Parent uses display: flex.
Key Properties
  • justify-content: align items horizontally
  • align-items: align items vertically
  • flex-direction: row, column
  • flex-wrap: wrap items
  • gap: spacing between items

CSS Grid

What is Grid?
CSS Grid is a powerful 2D layout system using display: grid. It allows defining rows and columns for precise control.
Grid Properties
  • grid-template-columns / rows
  • grid-gap
  • grid-column / row (child)
  • place-items, justify-items, align-items

Typography

Font Properties
  • font-family
  • font-size
  • font-weight
  • line-height
  • letter-spacing

Colors

Color Values
  • color — text color
  • background-color
  • Formats: named (e.g. red), hex (#ff0000), rgb, rgba, hsl, hsla

Media Queries

What Are Media Queries?
Media queries let you apply styles conditionally based on screen size, resolution, or orientation.
Example: @media (max-width: 600px) { body { font-size: 14px; } }

CSS Variables

Custom Properties
CSS variables are declared with --name and used with var().
Example: :root { --main-color: #333; }
body { color: var(--main-color); }
